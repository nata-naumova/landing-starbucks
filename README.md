# aroma-mini-store

### Структура проекта и базовая разметка:
1. Верстка страниц
    1.1 Главная
    1.2 Каталог
    1.3 Корзина
    1.4 Оформление заказа
    1.5 Детальная страница карточки товара

2. Маршрутизация между страницами

### Минимальная функциональность карточек товаров
1. Карточка товара
2. Детальная страница карточки
3. Моковые данные (JSON/API)

### Корзина
1. Мини-корзина (попап)
2. Добавление товара в корзину
3. Счетчик товаров в корзине
4. Сохранение данных в localStorage
5. Удаление товара из корзины

### Оформление заказа
1. Список товаров и итоговая сумма
2. Форма ввода контактных данных и адреса
3. Выбор способа доставки и оплаты
4. Кнопка "Подтвердить заказ", по нажатию:
    4.1 Вывод данных в консоль
    4.2 Уведомления об успешной операции
    4.3 Очистка корзины

### Фильтры, сортировка, поиск
1. Фильтры по типу кофе, ображке, цене
2. Сортировка по цене, популярности, новизне
3. Поиск товаров по названию

### Слайдер и отзывы
1. Слайдер с gsap
2. Товары с моковыми данными

### UX/UI
1. Кнопка "Наверх"
2. Модалки для карточек товаров
3. Уведомления для успешных действий (добавление в корзину, оформление заказа)

### Тестирование и полировка
1. Проверка орфограции, валидации форм
2. Кроссбраузерность и адаптивность
3. Финальный рефакторинг кода

### Документация и коммиты
1. Введение README с описанием страниц, функциональности и API
2. Все этапы разработки фиксируются отдельными коммитами
